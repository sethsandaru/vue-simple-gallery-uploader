# Vue Simple Uploader - Gallery Uploader

A simple UI for Vue to upload a gallery (images) or files with drag & drop for position arrangement

You can upload:
- Images (had preview mode)
- Files

## Built with
- JavaScript 
- Vue 2
- Sortable
